# Prescripto (Fullstack)

This workspace contains a React frontend (Create React App) and a minimal Express + MongoDB backend scaffold.

Project status (local):
- Frontend build: PASS (production build compiles with only minor ESLint warnings)
- Tests: PASS (unit smoke test updated to match the app)
- Backend: seed script and fallback file DB available; MongoDB Atlas supported when `MONGODB_URI` is configured and your IP is whitelisted.


Quick start (frontend)
1. Install dependencies at project root:

```powershell
cd c:\Users\Admin\prescripto
npm install
```

2. Run the frontend only:

```powershell
npm start
```

Backend (server)
- A `server/` folder exists with an Express + Mongoose app, seed script, and README (see `server/README.md`).
- Before starting the server you must create `server/.env` from `server/.env.example` and set `MONGODB_URI` (your MongoDB Atlas connection string) and `JWT_SECRET`.

To run both frontend and backend concurrently (dev):

```powershell
# from project root
npm run dev
```

Notes
- The frontend proxies API requests to `http://localhost:5000` (see `package.json` proxy setting). Update if you change the backend port.
- The project uses JWT auth for demo purposes and stores the token in localStorage (prototype only).

If you run into issues, paste the server log output here and I will help debug.
